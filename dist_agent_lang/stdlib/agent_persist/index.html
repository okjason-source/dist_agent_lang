Module agent_persist 

Source
Expand description

Persistent agent runtime: snapshot, backends (file, SQLite), config. Persistence is ON by default; set DAL_AGENT_RUNTIME_PERSIST=0 or agent.toml [agent] runtime_persist = false to disable. Backend: DAL_AGENT_RUNTIME_BACKEND=file|sqlite (default: file). Path: DAL_AGENT_RUNTIME_PATH or agent.toml [agent] runtime_path.
