Agent skills: extensible skill registry, .skill.dal loader, and prompt builder. See docs/SKILLS_DESIGN_PLAN.md.

Skills are named bundles (description, tools) that define what an agent can do. Built-in skills: development, creative, office, home, project_init. User-defined skills are loaded from .skill.dal files at a configurable path. Persistence is the default for the registry: loaded once at first use.
