Agent skills: built-in skill definitions and tools_description builder. See docs/AGENT_ASSISTANT_PLAN.md §3.

Skills are named bundles (description, tools) that define what an agent can do. Default learning-path skills: development, creative, office, home. Hard skill: project_init (every agent can init a DAL project).

Constants§

DEFAULT_LEARNING_PATH_SKILLS
Default skill set for the DAL assistant (smart agent assist). When an agent has no explicit skills (e.g. no mold or mold has no skills), use these.
PROJECT_INIT_SKILL
Hard skill: always available to every agent. Used when building tools description.

Functions§

tools_description_for_skills
Build the tools_description string for the agent prompt from the given skill names. Always includes project_init (hard skill). If skills is empty, uses DEFAULT_LEARNING_PATH_SKILLS. Development skill adds read_file, write_file, list_dir, dal_check, dal_run.
