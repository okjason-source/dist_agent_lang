pub fn tools_description_for_skills(skills: &[String]) -> String
Expand description

Build the tools_description string for the agent prompt from the given skill names. Uses the global registry. Always includes project_init (hard skill) and encouragement. If skills is empty, uses DEFAULT_LEARNING_PATH_SKILLS. This is the main entry point used by agent_serve.rs.
