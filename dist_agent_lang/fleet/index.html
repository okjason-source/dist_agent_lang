Module fleet 

Source
Expand description

Fleet: named set of agents, optionally created from a mold (N instances). Off-chain only. Storage: in-memory; when base path is provided, also persisted to .dal/fleets.json. See COMPREHENSIVE_AGENT_AND_MOLD_PLANS.md §5.

Functions§

add_from_mold
Add N members to an existing fleet from a mold. If the fleet is empty, sets mold_source. Use this to populate an empty fleet or add more members (same as scale-up logic but explicit).
add_member
Add a single agent (by ID) to an existing fleet. Agent must already exist elsewhere; this only registers the ID.
create
Create an empty fleet with the given name. Fails if name already exists. If base is Some, fleet data is persisted to base/.dal/fleets.json.
create_from_mold
Create a fleet by spawning N agents from a mold. Same principal/mold/fallback rules as single create. Each agent is named <fleet_name>_0, <fleet_name>_1, …; params apply to each. Fleet is persisted to base/.dal/fleets.json.
delete
Remove a fleet (agents remain in runtime; only the grouping is removed). Returns true if removed. If base is Some, persists after remove.
deploy
Record a deployment (task) for a fleet. Updates last_deployed_task and last_deployed_at; persists if base is set. Runners/automation can read the fleet file to see what to execute for each fleet.
export
Export fleet(s) to YAML (k8s Job list or docker-compose). If name is None, exports all fleets.
health
Report health for a fleet. Status is “ok” if fleet exists and has at least one member or a mold; “empty” if no members and no mold.
list
List fleet names. If base is Some, loads from base/.dal/fleets.json first.
run
Run fleets: for each fleet with last_deployed_task (and name matching filter if given), ensure it has members (if empty and has mold_source, add one from mold), then dispatch last_deployed_task to each member via agent::coordinate(agent_id, task, “task_distribution”). Returns one RunReport per fleet processed.
scale
Resize fleet to N members. If base is Some, persists after change.
show
Get fleet by name, if it exists. If base is Some, loads from file first.
