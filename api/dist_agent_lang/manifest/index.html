add_dependencies_if_missing
Add package names to [dependencies] in dal.toml if not already present (with given version). Returns the number of entries added. Preserves existing keys; new keys use version (e.g. “latest”).
load_resolved_deps
Read dal.lock and return name -> path. If lockfile missing, resolve from manifest and return. When a cached path is missing, re-fetches from [metadata] source if available.

resolve_dependencies
Resolve dependencies: path deps canonicalized; version deps fetched from registry and cached. Returns (resolved deps, version metadata for lockfile).
write_lockfile
Write dal.lock next to manifest_path with resolved paths and optional version metadata.
