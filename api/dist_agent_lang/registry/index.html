Fetch a package from source (HTTP/HTTPS URL or ipfs://<cid>), unpack to a temp dir, return the unpacked path.
find_package_root
Find package root (directory containing dal.toml) within an unpacked path. Tarballs often have a single top-level dir (e.g. package-1.0.0/); this locates dal.toml.
pack_package
Pack project at manifest_path into a gzipped tarball. Includes dal.toml and all *.dal under project root. Tarball has a single top-level directory {name}-{version}/.
publish_package
Publish the package at manifest_path to the registry. Requires DAL_REGISTRY_TOKEN. Uses http-interface.
