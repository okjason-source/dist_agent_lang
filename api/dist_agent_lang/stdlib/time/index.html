Module time

Wall-clock and duration helpers for DAL (no LLM). Use with crate::stdlib::scatter for relative (scatter::after_ms) vs absolute (scatter::after_at_unix_ms) scheduling.

All Unix times are milliseconds since 1970-01-01T00:00:00Z as i64 (DAL Int). RFC3339 strings include timezone offset or Z.
