Agent that can reply, run shell commands, or search the web. Runs a multi-step loop until the LLM returns a final reply (or ask_user / parse_fail / max steps). So after a “run” (e.g. curl to post a tweet), the agent gets the tool result and continues until it sends a user-facing reply. When scripting env is enabled (see scripting_env_enabled), exposes write_file, read_file, list_dir, dal_run, dal_check and uses scripts/ under the configured root when set.
