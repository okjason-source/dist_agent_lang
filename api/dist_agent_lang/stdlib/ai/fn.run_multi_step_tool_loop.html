Expand description

Run the tool loop until the LLM returns reply or ask_user, or max_steps is reached. Appends each run/search to evolve action log when agent_name is Some. working_root: if Some, file tools (read_file, write_file, list_dir, dal_check, dal_run, dal_init) use this path; else process current_dir (Phase D). Caller should append_conversation(user_msg, result.final_text) when done.
