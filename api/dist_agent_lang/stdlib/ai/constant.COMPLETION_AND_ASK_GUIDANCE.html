Expand description

Completion and when to ask: try to finish; if you need input or must stop, keep the user in the loop. Default copy favors multi-step work (long tasks, scaffolding, many runs). Override entirely with env DAL_AGENT_COMPLETION_AND_ASK_GUIDANCE (non-empty string). Public for IDE agent runner (prefer completion_and_ask_guidance_for_tool_loop so env is honored).
