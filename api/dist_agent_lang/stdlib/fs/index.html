Module fs

Host filesystem access from DAL (fs::read_text, fs::write_text, etc.).

Paths are relative to a root directory: process current directory by default, or DAL_FS_ROOT when set (canonicalized when possible). .., absolute paths, and Windows UNC prefixes are rejected (same jail as agent read_file / write_file).
