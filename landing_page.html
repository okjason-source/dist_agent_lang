Build from Source

Recommended for development and latest features

# Clone repository git clone https://github.com/okjason-source/dist_agent_lang.git cd dist_agent_lang # Build cargo build --release # Test the binary ./target/release/dal --version # Add to PATH (optional) export PATH="$PWD/target/release:$PATH"

Requirements: Rust 1.70+ (install from rustup.rs)

Linux / macOS

Download pre-built binary

# Download latest release for your platform # Linux x64: wget https://github.com/okjason-source/dist_agent_lang/releases/latest/download/dist_agent_lang-v1.0.5-linux-x64.tar.gz # macOS Intel: wget https://github.com/okjason-source/dist_agent_lang/releases/latest/download/dist_agent_lang-v1.0.5-macos-x64.tar.gz # macOS Apple Silicon: wget https://github.com/okjason-source/dist_agent_lang/releases/latest/download/dist_agent_lang-v1.0.5-macos-arm64.tar.gz # Extract tar -xzf dist_agent_lang-v1.0.5-*.tar.gz # Install (or use directly) sudo cp dal /usr/local/bin/ # Verify dal --version

Windows

Download pre-built binary

# Download latest release # Download: dist_agent_lang-v1.0.5-windows-x64.zip # Extract the zip file # Open Command Prompt in extracted folder # Copy dal.exe to a folder in your PATH # (e.g., C:\Program Files\dist_agent_lang\) # Add that folder to your system PATH # Verify dal --version

Homebrew (macOS)

Install via Homebrew package manager

# Add tap (when available) brew tap distagentlang/dist_agent_lang # Install brew install dist_agent_lang # Verify dist_agent_lang --version

Note: Homebrew package coming soon

Install

cargo install dist_agent_lang

Running the above command will globally install the dal binary.

Verify: dal --version

Install as library

Run the following Cargo command in your project directory:

cargo add dist_agent_lang

Or add the following line to your Cargo.toml:

dist_agent_lang = "1.0.5"

npm (Node.js)

Install via npm package manager

# Install globally npm install -g dist-agent-lang # Verify dist_agent_lang --version

Note: npm package coming soon

üå± Quick Start

Get up and running in 5 minutes

Step 1: Verify Installation

dal --version # Should output: # dist_agent_lang v1.0.5 # Beta Release - Actively Developed

Step 2: Create Your First Program

Create a file hello.dal:

@trust("hybrid") service HelloWorld { fn main() { print("Hello, dist_agent_lang!"); } }

Note: Make sure you're in the correct directory when creating files.

Step 3: Run It

dal run hello.dal # Output: Hello, dist_agent_lang!

Tip: Check out the examples/ directory in the repository for more examples!
