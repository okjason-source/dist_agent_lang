## Contribution Guidelines

### Adding Yourself

When your contribution is merged:
1. You will be automatically added to this list (if using automation)
2. Or you can add yourself in a PR
3. Include your name and a link to your GitHub profile (optional)
4. Categorize yourself appropriately

### Format

```markdown
- [Your Name](https://github.com/yourusername) - Brief description of contribution
```

### Categories

- **Code Contributors**: PRs with code changes
- **Documentation Contributors**: Documentation improvements
- **Testing Contributors**: Test additions, bug reports, testing feedback
- **Bug Reporters**: Reported bugs that led to fixes
- **Feature Suggesters**: Feature requests that were implemented
- **Community Helpers**: Helpful discussions, answers, community support
